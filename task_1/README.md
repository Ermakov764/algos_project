# Task 1: Реверс односвязного списка

## Описание

Реализация разворота односвязного списка **in-place** методом трёх указателей. Классическая задача интервью по алгоритмам.

## Алгоритм

- **Метод**: три указателя `prev`, `current`, `next_node`
- На каждой итерации меняем направление связи: `current.next = prev`
- Сдвигаем указатели вперёд по списку
- Результат — новый «голова» списка в `prev`

## Сложность

| Метрика | Значение |
|---------|----------|
| Время | O(n) |
| Память | O(1) |

## Файлы

| Файл | Описание |
|------|----------|
| `algorithm.py` | Класс `ListNode`, функция `reverse_linked_list`, утилиты |
| `visualization.py` | Интерактивная демо с замерами времени и памяти |

## Запуск

```bash
python visualization.py
```

Режимы:
1. **Ручной ввод** — числа через пробел (например: `1 2 3 4 5`)
2. **Генерация** — выбор размера: 10, 50 или 100 элементов

Программа выводит исходный список, результат реверса, время выполнения и потребление памяти.
