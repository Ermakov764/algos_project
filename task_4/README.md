# Task 4: Максимальная прибыль при k сделках

## Описание

Задача о торговле акциями: дан массив цен по дням, разрешено не более **k** сделок (покупка + продажа = 1 сделка). Требуется найти максимальную прибыль.

## Алгоритм

**Динамическое программирование:**
- `hold[t]` — макс. прибыль, если держим акцию после t-й сделки
- `sold[t]` — макс. прибыль после завершения t сделок (не держим)

**Оптимизация:** при `k ≥ n/2` ограничение несущественно — жадный подход: сумма всех положительных разностей цен между соседними днями.

## Сложность

| Случай | Время | Память |
|--------|-------|--------|
| k < n/2 | O(n·k) | O(k) |
| k ≥ n/2 | O(n) | O(1) |

## Запуск

```bash
python task_4.py
```

Пример ввода:
```
Цены: 3 2 6 5 0 3
k: 2
```

Результат: максимальная прибыль при ограничении на число сделок.
