# Task 3: Красно-чёрное дерево

## Описание

Реализация **красно-чёрного дерева** (Red-Black Tree) — самобалансирующегося двоичного дерева поиска, которое гарантирует высоту O(log n) при любых вставках.

## Свойства КЧ-дерева

1. Каждый узел — либо красный, либо чёрный
2. Корень — всегда чёрный
3. Листья (NIL) — чёрные
4. Красный узел не может иметь красного родителя
5. Любой путь от корня до листа содержит одинаковое число чёрных узлов

## Особенности реализации

- Поддержка **дубликатов** (счётчик в узле)
- Работа с **0** и **отрицательными** числами
- Итеративная балансировка (без рекурсии, O(1) доп. памяти)
- Левый/правый повороты и перекраска при вставке

## Сложность

| Операция | Время | Память |
|----------|-------|--------|
| Вставка | O(log n) | O(1) доп. |
| In-order обход | O(n) | O(n) стек |

## Запуск

```bash
python task_3.py
```

Вводите целые числа по одному. Для завершения введите `stop`. Программа показывает текущее состояние дерева после каждой вставки.
