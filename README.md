# Алгоритмы и структуры данных

Коллекция реализаций классических алгоритмов и структур данных на Python с подробной документацией, анализом сложности и интерактивными демонстрациями.

| [Task 1](task_1/) | Реверс односвязного списка (метод трёх указателей) | O(n) время, O(1) память |
| [Task 2](task_2/) | Ранжирование объектов по релевантности | O(d log d) / O(1) |

## Описание задач

### Task 1: Реверс односвязного списка

Классическая задача: развернуть односвязный список in-place методом трёх указателей. Включает:
- Измерение времени выполнения и потребляемой памяти (`tracemalloc`)
- Интерактивный режим: ручной ввод или генерация списка

### Task 2: Ранжирование по релевантности

Объекты с признаками, релевантность = взвешенная сумма признаков. Запросы:
- **Тип 1**: вывести k наиболее релевантных объектов
- **Тип 2**: обновить признак объекта (инкрементальный пересчёт)

### Task 3: Красно-чёрное дерево

Самобалансирующееся двоичное дерево поиска:
- Поддержка дубликатов (счётчик в узле)
- Работа с 0 и отрицательными числами
- Левый/правый повороты и перекраска при вставке

### Task 4: Максимальная прибыль при k сделках

Динамическое программирование для торговли акциями:
- Не более k сделок (покупка + продажа = 1 сделка)
- Оптимизация: при k ≥ n/2 — жадный подход

### Task 5: День с максимальным числом гостей

Sweep line / метод событий:
- События: заезд (+1), отъезд (-1 на следующий день)
- Сортировка событий и сканирование с поддержкой счётчика

### Task 6: Алгоритм Беллмана–Форда

Кратчайшие пути в графе:
- Поддержка отрицательных весов
- Обнаружение отрицательных циклов
- Пошаговая визуализация релаксации рёбер
